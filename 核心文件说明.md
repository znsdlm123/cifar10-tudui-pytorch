# CIFAR-10 图像分类项目 - 核心文件列表

## 核心文件（完成项目必需）

### 1. 模型定义
- **`model.py`** - 定义了 Tudui CNN 模型类（项目核心模型）

### 2. 训练脚本
- **`train.py`** - CPU 训练脚本（主训练文件）
- **`train_gpu_1.py`** - GPU 训练脚本（使用 .cuda() 方法）
- **`train_gpu_2.py`** - GPU 训练脚本（使用 .to(device) 方法，推荐）

### 3. 数据处理
- **`dataset_transform.py`** - 数据集加载和可视化
- **`dataloader.py`** - 数据加载器测试和可视化

### 4. 模型管理
- **`model_save.py`** - 模型保存示例
- **`model_load.py`** - 模型加载示例
- **`model_pretrained.py`** - 预训练模型使用示例

### 5. 模型推理
- **`src/test.py`** - 使用训练好的模型进行图像分类推理

## 快速开始

### 最小运行配置（只需 3 个文件）
1. `model.py` - 模型定义
2. `train.py` - 训练脚本
3. `src/test.py` - 推理测试（可选）

### 完整项目配置（推荐）
- 使用 `model.py` + `train_gpu_2.py` 进行训练
- 使用 `src/test.py` 进行推理
- 使用 `dataset_transform.py` 和 `dataloader.py` 进行数据验证

## 文件依赖关系

```
train.py
  ├── model.py (导入 Tudui 类)
  └── 数据集: ./data/

train_gpu_1.py / train_gpu_2.py
  ├── 模型定义（内嵌）
  └── 数据集: ./data/

src/test.py
  ├── 模型文件: tudui_0.pth
  └── 测试图像: ../img/

dataset_transform.py
  └── 数据集: ./dataset/

dataloader.py
  └── 数据集: ./dataset/
```
