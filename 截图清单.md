## 🚀 快速执行顺序

### 推荐顺序（高效）：

1. **先完整训练一次**（最耗时，可以放后台）
   ```bash
   python train_gpu_2.py
   ```
   ⏱️ 预计时间：10-30分钟（取决于GPU/CPU）

2. **训练同时或之后，准备其他截图：**
   ```bash
   # 数据可视化（快速，1-2分钟）
   python dataloader.py
   tensorboard --logdir=dataloader
   
   # 推理测试（快速，几秒）
   python src/test.py
   ```

3. **训练完成后，查看训练曲线：**
   ```bash
   tensorboard --logdir=logs
   ```
   ⏱️ 查看和截图：5-10分钟

4. **最后截图：**
   - 项目结构（文件管理器）
   - 模型代码（编辑器）
   - 模型文件列表（文件管理器）

## 💡 截图技巧

1. **TensorBoard 截图：**
   - 使用浏览器全屏或调整窗口大小
   - 可以隐藏左侧导航栏（点击折叠按钮）
   - 确保曲线清晰可见

2. **终端截图：**
   - Windows：使用 `Win + Shift + S` 快速截图
   - 或使用终端工具（如 Windows Terminal）的清晰字体
   - 确保文本清晰可读

3. **代码截图：**
   - 使用代码编辑器（PyCharm/VSCode）的截图功能
   - 或直接复制代码块到报告（Markdown 代码块格式）

4. **图片拼接：**
   - 可以使用 PowerPoint、Photoshop 或在线工具拼接多张截图
   - 或者直接在报告中并排放置
